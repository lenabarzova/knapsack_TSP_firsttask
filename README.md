Теперь в моей программе имя входного файла передаётся через командные аргументы. Это сделано для того, чтобы не изменять код каждый раз при проверке нового теста.
knapsack.cpp - файл с реализацией чтения входных данных, алгоритма решения задачи и вывода результата
knapsack.h - заголовочный файл с описанием структур и функций
main.cpp - точка входа в программу. В этом файле происходит получение имени входного файла из командной строки, чтение данных и вызов алгоритма решения задачи
